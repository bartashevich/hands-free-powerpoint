Introduction

Sometimes, we need to extract text from PowerPoint Presentation to other applications like NotePad or Microsoft Word to reduce its size. This sample is aimed to give a C# solution to save text from PowerPoint to .txt or .doc.

Description

Tools We Need:

- Free Spire.Presentation.dll
- Visual Studio

Detailed Steps:

Step 1: Create or open a .NET class application in Visual Studio 2005 or above versions. Add Spire.Presentation.dll to your .NET project assemblies.

Step 2: Create a new instance of Spire.Presentation class and load the sample PowerPoint file based on specified path.

Step 3:Initialize a new instance of StringBuilder class, append extracted text from Presentation to StringBuilder.

Step 4: Create a new .txt or .doc file and write in string text using  File.WriteAllText(string path, string contents).

Presentation presentation = new Presentation("test.ppt", FileFormat.PPT);
            StringBuilder sb = new StringBuilder();

            foreach (ISlide slide in presentation.Slides)
            {
                foreach (IShape shape in slide.Shapes)
                {
                    if (shape is IAutoShape)
                    {
                        foreach (TextParagraph tp in (shape as IAutoShape).TextFrame.Paragraphs)
                        {
                            sb.Append(tp.Text + Environment.NewLine);
                        }
                    }

                }

            }
            File.WriteAllText("result.txt", sb.ToString());
